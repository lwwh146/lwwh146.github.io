# 体重饮食记录应用

一个基于Vue 3的移动端体重和饮食记录应用，使用ECharts展示体重趋势图表，数据存储在本地localStorage中。

## 功能特性

- 📱 移动端优化设计
- ⚖️ 体重记录和管理
- 🍽️ 饮食记录（早餐、午餐、晚餐、加餐）
- 📊 体重趋势图表展示
- 📈 统计数据分析
- 💾 本地数据存储
- 🎨 现代化UI设计

## 技术栈

- Vue 3
- Vue Router
- ECharts
- Vite
- localStorage

## 项目结构

```
src/

│   ├── DietRecord.vue     # 饮食记录页面

```

## 使用说明

### 1. 体重记录
- 点击底部导航的"体重"标签
- 选择日期，输入体重数值
- 可选添加备注信息
- 点击"添加记录"保存

### 2. 饮食记录
- 点击底部导航的"饮食"标签
- 选择日期和餐次（早餐/午餐/晚餐/加餐）
- 输入食物名称和卡路里（可选）
- 可选添加备注信息
- 点击"添加记录"保存

### 3. 查看图表
- 点击底部导航的"统计"标签
- 选择时间范围（7天/30天/3个月/6个月/1年）
- 查看体重趋势图表和统计数据
- 浏览详细的历史记录

### 4. 首页概览
- 查看最新体重和今日餐数
- 浏览最近的记录
- 快速了解整体情况

## 数据存储

应用使用浏览器的localStorage存储数据，包括：
- 体重记录：日期、体重、备注、创建时间
- 饮食记录：日期、餐次、食物、卡路里、备注、创建时间

### 数据备份与恢复

- **导出数据**：在数据管理页面点击"导出数据"，将所有数据保存为JSON文件
- **导入数据**：选择之前导出的JSON文件，支持合并模式或覆盖模式
- **数据格式**：标准JSON格式，包含版本信息和导出时间戳
- **跨设备同步**：通过导出/导入功能实现数据在不同设备间的迁移

建议定期导出数据进行备份，避免数据丢失。
